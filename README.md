# RRightclickrr

Right click any file to upload it to Google Drive.

## Features

- **One-click upload** - Right-click â†’ Upload to Google Drive
- **Progress indicator** - See upload status in system tray
- **Auto-copy link** - Share link copied to clipboard automatically
- **Native integration** - Works directly from Windows Explorer

## Installation

Download the latest release from [Releases](https://github.com/Kepners/rrightclickrr/releases).

Run the installer and sign in with your Google account.

## Usage

1. Right-click any file in Windows Explorer
2. Click "Upload to Google Drive"
3. Wait for upload (progress shown in system tray)
4. Share link is automatically copied to clipboard!

## Development

```bash
# Install dependencies
npm install

# Run in development
npm run dev

# Build for production
npm run build
```
