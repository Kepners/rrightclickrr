RRightclickrr

Sync any folder to Google Drive with a simple right-click. Free and open-source.

Download for Windows View on GitHub

Features

Right-Click to Sync

Just right-click any folder and select "Sync to Google Drive". No drag-and-drop, no browser tabs.

Auto-Sync on Changes

Set up folder mappings and your files sync automatically when they change. Set it and forget it.

Exclude Subfolders

Don't need to sync node_modules or that huge assets folder? Exclude specific subfolders to save bandwidth.

Secure OAuth

Sign in with your Google account securely. Your credentials never touch our servers.

Instant Share Links

After upload, the Google Drive link is automatically copied to your clipboard. Share instantly.

System Tray

Runs quietly in the background. Quick access to settings and recent syncs from the system tray.

How It Works

1

Install & Sign In

Download, install, and sign in with your Google account.

2

Enable Right-Click Menu

Click "Enable Right-Click Menu" in the app settings.

3

Right-Click Any Folder

Right-click a folder → "Show more options" → "Sync to Google Drive".
